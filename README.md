# &lt;api-viewer&gt;

API viewer for [web-component-analyzer](https://github.com/runem/web-component-analyzer) JSON output.

## Usage

```html
<api-viewer json-src="./custom-elements.json"></api-viewer>
```

## Install dependencies

```sh
yarn
```

## Run demo in browser

```sh
yarn dev
```

Open http://127.0.0.1:8081/demo/
